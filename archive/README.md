# Archive

Old experiment scripts from the generic "ReFine-Lab learning hub" phase.
These are not part of the RV-GRPO research pipeline.

Kept for reference, not used in any Makefile target or paper experiment.

## Contents

- `experiments/dpo/train_dpo.py` — old DPO script (replaced by `experiments/alignment/train_dpo.py`)
- `experiments/ppo/train_ppo.py` — PPO trainer (not used in RV-GRPO pipeline)
- `experiments/rlhf/pipeline.py` — full RLHF pipeline (not used)
- `experiments/train_lora.py` — generic LoRA script (replaced by `experiments/sft/train_sft.py`)
- `experiments/train_simpo.py` — SimPO trainer (not used)
- `experiments/run_experiments.sh` — old batch runner (replaced by Makefile)
